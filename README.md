# @lbwa/tsconfig

Shared TypeScript config.

## Installation

```bash
yarn add @lbwa/tsconfig -D
# or npm i @lbwa/tsconfig -D
```

## Usage

In your project root `tsconfig.json`:

```json
{
  "extends": "@lbwa/tsconfig",
  "compilerOptions": {
    "outDir": "dist"
  }
}
```
